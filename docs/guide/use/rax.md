# 用 Rax 开发一个 Kraken 应用

## 初始化项目

基于 `npm init rax` 命令，可以快速创建一个 Rax 多端应用（注意：npm 版本需 >= 6.1.0）：

```shell
$ npm init rax <YourProjectName>
```

初始化项目过程中， 请依次选择 `App`、 `Kraken 跨端应用`，然后根据你的喜好选择 `JavaScript` 或者 `TypeScript`。

按流程创建出前端工程目录后启动 dev server：

```shell
$ cd <YourProjectName>
$ npm install
$ npm start
```
